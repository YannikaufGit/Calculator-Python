A desktop calculator built with Python and Pygame to demonstrate :highlight[object-oriented programming], expression parsing, and clean project structure.

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- Correct operator precedence
- Unary minus support (e.g. `5*-2`)
- Decimal numbers
- Error handling
- Modern dark UI

## Structure
- `engine.py` → Calculation logic
- `app.py` → Application loop & rendering
- `button.py` → UI components
- `main.py` → Entry point

## Run

```bash
python main.py
